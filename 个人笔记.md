# webpack中文件处理的两个插件file-loader和url-loader

## file-loader
> 主要作用是将执行的文件生成到打包目录中，并返回文件地址

```js
function loader() {
  // 使用commonjs进行导入，导出格式如下
  return 'module.exports=文件地址'

  // 使用es6进行导入， 导出格式如下
  return '文件地址'
}
```


## url-loader
> 主要作用是将项目中的文件转化成base格式, 可以指定大小的文件来决定是返回base64格式编码还是文件地址

```js
function loader() {
  return 'module.exports=base64格式编码'
}
```
